# to-do-list
A powerful and aesthetic study companion built to transform the way students focus and manage their time. By blending task management, focus tracking, and smart insights, this app creates an immersive and motivating environment that encourages productivity while keeping the experience smooth, colorful, and enjoyable.
Study Companion – Focus, Track & Grow

A modern, colorful, and user-friendly productivity web app designed to help students stay focused, manage tasks, and build better study habits.

🚀 Features
⏳ Focus Timer
Countdown timer with start, pause, and reset
Helps implement Pomodoro technique
Smooth and interactive UI
😵‍💫 Distraction Tracker
Track interruptions during study sessions
Real-time counter updates
Helps improve focus awareness
📋 Task Management System
Add, edit, and delete tasks
Categorize tasks (Work, Study, Personal, etc.)
Set due dates
Filter tasks (All / Active / Completed)
Sort tasks based on priority or date
📊 Smart Dashboard
Displays:
Total tasks
Active tasks
Overdue tasks
Dynamic updates with animations
🎨 Modern UI/UX
Colorful gradient design
Clean card-based layout
Responsive design (mobile + desktop)
Smooth transitions and hover effects
💾 Data Persistence
Uses localStorage to save tasks and progress
Data remains even after page refresh
🧠 Technologies Used
HTML5
CSS3
JavaScript (Vanilla JS)
LocalStorage API
🎯 Objective

To create an interactive and visually appealing productivity tool that helps users:

Stay organized
Improve focus
Manage time effectively
Build consistent study habits
